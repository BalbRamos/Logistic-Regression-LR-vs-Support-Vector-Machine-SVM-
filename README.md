# Adult Income Classification Project

## Project Title
Adult Income Classification using Logistic Regression and SVM

## Author
Balbino Souza Ramos Neto

## Research Paper Summary
This project is based on *Supervised Learning for Binary Classification on US Adult Income* by Li-Pang Chen, published in 2021 (https://www.xpublication.com/index.php/jmo/article/download/609/332/).   
The paper studies binary income classification using the Adult income dataset (https://archive.ics.uci.edu/dataset/2/adult) and compares methods including Logistic Regression and SVM.
For this DS 301 project, the implementation focuses only on class-approved models from that setting: Logistic Regression and Support Vector Machine. 

## Dataset Description
The project uses the Adult / Census Income dataset from the UCI Machine Learning Repository.
The dataset is a classification dataset with 48,842 instances and 14 features.
The task is to predict whether an individual's annual income exceeds $50K based on demographic and work-related attributes. 
The dataset contains both categorical and integer variables and includes missing values, which makes it suitable for preprocessing and classification analysis.
